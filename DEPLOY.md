# วิธีการนำเว็บไซต์ขึ้น Server ฟรี (Render.com)

เพื่อให้ระบบสามารถทำงานได้สมบูรณ์ (สมัครสมาชิกได้, บันทึกข้อมูลได้) ต้องนำ Code ไปรันบน Server จริงครับ โดยแนะนำ **Render.com** เพราะฟรีและทำได้ง่าย

## ขั้นตอนที่ 1: เตรียม Code ให้พร้อม (ทำเสร็จแล้ว)
ผมได้ทำการแก้ไขไฟล์ `server.js` ให้รองรับการทำงานบน Server เรียบร้อยแล้ว (แก้ Port เป็น Dynamic และตั้งค่าให้หน้าแรกเป็น `indexApp.html`)

## ขั้นตอนที่ 2: สมัครและเชื่อมต่อ Render
1. เข้าไปที่ [https://render.com/](https://render.com/) และกดปุ่ม **GET STARTED** (สมัครด้วย GitHub จะง่ายที่สุด)
2. เมื่อเข้าสู่หน้า Dashboard ให้กดปุ่ม **New +** แล้วเลือก **Web Service**
3. เลือก **Build and deploy from a Git repository**
4. เชื่อมต่อกับ GitHub ของคุณ และเลือก Repository: `my-crypto-app`
5. กดปุ่ม **Connect**

## ขั้นตอนที่ 3: ตั้งค่า Web Service
กรอกข้อมูลดังนี้:
- **Name:** ตั้งชื่อแอปของคุณ (เช่น `my-crypto-tycoon`)
- **Region:** Singapore (ใกล้ไทยที่สุด เร็วสุด)
- **Branch:** `main`
- **Root Directory:** (เว้นว่างไว้)
- **Runtime:** `Node`
- **Build Command:** `npm install`
- **Start Command:** `node server.js`
- **Plan:** เลือก **Free**

6. กดปุ่ม **Create Web Service** ด้านล่างสุด

## ขั้นตอนที่ 4: รอระบบทำงาน
- Render จะเริ่มดึง Code และติดตั้งระบบ (ใช้เวลาประมาณ 1-3 นาที)
- เมื่อเสร็จแล้ว จะขึ้นสถานะสีเขียวว่า **Live**
- คุณจะได้รับ URL ของเว็บ (เช่น `https://my-crypto-tycoon.onrender.com`)

## ขั้นตอนที่ 5: ทดสอบใช้งาน
- คลิกที่ลิงก์ URL ที่ได้จาก Render
- ลองสมัครสมาชิกและใช้งานระบบดูครับ

**ข้อควรระวัง:**
- ข้อมูลบน Server ฟรีอาจจะหายไปถ้ามีการ Restart เครื่อง (เพราะเราใช้ไฟล์ JSON เก็บข้อมูล ไม่ได้ใช้ Database จริง) แต่สำหรับการทดสอบถือว่าเพียงพอครับ
- เว็บฟรีของ Render จะ "หลับ" (Sleep) ถ้าไม่มีคนเข้าใช้งานสักพัก เมื่อเข้าใหม่ครั้งแรกอาจจะช้าหน่อย (รอประมาณ 30-50 วินาที)
